How do I get started with RBLN SDK? ‚ûï
To get started with RBLN SDK, download RBLN driver, compiler and Model Zoo in appropriate versions.
  • Install RBLN Driver.
  • Install RBLN Compiler.
  • Check whether RBLN SDK supports your desired model and check the code in Model Zoo.
  • Run the sample code to ensure there are no issues.
For a detailed installation guide, please refer to SDK Installation Guide.
How do I check if my model is compatible? ‚ûï
Please refer to the Model Zoo documentation to check supported models. If your model is not listed, you can request compatibility verification through our support team.
What is the recommended environment for RBLN SDK? ‚ûï
The recommended development environment includes Ubuntu 20.04 or higher, Python 3.8+, and GCC 9.0+. Make sure all dependencies are correctly installed.
Can I deploy on multiple GPUs? ‚ûï
Yes, RBLN SDK supports multi-GPU deployment using tensor parallelism. Refer to the scaling documentation for details.
How do I report a bug? ‚ûï
If you encounter any bugs, please submit a report through our support portal or create an issue in our GitHub repository.
